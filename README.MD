# 🎓 Campus Placement Portal

A full-stack web application that connects students with companies for campus placements. Built with transparency in mind — students can see vacancy count and applicant count on every job listing.

## 🚀 Live Demo
Coming soon...

## ✨ Features
- Student signup/login and job browsing
- Company signup/login and job posting
- Real-time applicant count on every listing
- Student application tracking (applied/shortlisted/rejected)
- Eligibility based job filtering (branch, CGPA, backlogs)
- Resume upload and auto generation
- Student profile with projects, certifications, internships

## 🛠️ Tech Stack
- **Frontend:** React + Tailwind CSS
- **Backend:** Node.js + Express
- **Database:** MongoDB Atlas + Mongoose
- **Auth:** JWT + Bcrypt
- **Storage:** Cloudinary
- **Deployment:** Vercel (frontend) + Render (backend)

